# Puppet Bootstrap

# Usage 
This will bootstrap a puppet installation on centos, Ubuntu, or MACOS.

To set up puppet on any of the supported platforms:

`sudo install.sh`

# Versions 

On Mac facter-1.6.3 will be loaded, hiera-1.2.0, and puppet-3.2.1 will be installed.
